## 💻 Getting Started <a name="getting-started"></a>

### Prerequisites

```sh
 gem install rails
```

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/kennankole/kennedy-antony-TDD.git
```
### Install

Install this project with:

```sh
  cd kennedy-antony-TDD
  gem install
```

### Usage

To run the project, execute the following command:

```sh
  ruby main.rb
```

### Run tests

To run tests, run the following command:


```sh
  rspec __tests__/
```
